 Simple AI Chatbot (Frontend)

A simple AI Chatbot frontend application that allows users to interact with an AI through a clean and user-friendly interface. This project focuses on building a responsive UI and handling real-time chat interactions.

---

##  Features

-  Real-time chat interface
-  Dynamic message rendering
-  Clean and responsive UI
-  API integration (for AI responses)
-  Mobile-friendly design

---

##  Tech Stack

- Frontend: React.js 
- Styling: CSS / Bootstrap 
- API: REST API / OpenAI API
